# 外卖配送运营与客户行为分析

## 项目背景
外卖平台面临配送延迟、订单取消等痛点，直接影响客户体验和平台营收。本项目基于 Kaggle 上的外卖配送脱敏数据集，分析配送效率的关键影响因素，识别高价值客户特征，为平台提供数据驱动的优化建议。

## 分析目标
- 定位导致**订单延迟**和**取消**的主要因素
- 对比不同**城市等级**、**会员状态**、**节假日**下的订单表现
- 探索**客户忠诚度**与**订单金额**的关系
- 提出可落地的业务改进策略

## 数据集说明
- **来源**：Kaggle（食品配送运营与客户分析数据集）
- **规模**：15,000 条订单记录，30 个字段
- **主要字段**：
  - 订单信息：订单金额、商品数量、取消/延迟标志
  - 配送信息：距离、备餐时间、配送时长、预计时长、交通/天气评分、配送员经验
  - 客户信息：年龄、忠诚度评分、是否会员
  - 评价信息：餐厅评分、配送员评分、客户评分
  - 外部因素：城市等级、节假日/周末标志、订单时段

> 注：数据已脱敏，不包含个人身份信息。

## 分析步骤与主要结论

### 1. 数据清洗与预处理
- 检查缺失值：`delivery_partner_rating`、`customer_rating`、`tip_amount` 缺失率均为 1%，使用均值/0 填充
- 无重复值，无需删除
- 构造新特征：`delivery_diff` = 实际配送时长 - 预计配送时长

### 2. 配送效率分析
- **整体延迟率**：9.47%
- **不同城市等级的平均配送时长**：
  - 城市等级 3（三级城市）配送时间最长，城市等级 1（一级城市）最短 → 配送效率与城市发展水平正相关

### 3. 取消订单特征对比
- 取消订单组的**配送距离**、**备餐时间**、**交通评分**（数值越高表示交通越差）均显著高于未取消组
- 表明：长距离、长备餐时间、恶劣交通状况会明显增加取消风险

### 4. 客户分层与价值分析
- **会员 vs 非会员**：会员客单价普遍高于非会员
- **忠诚度评分与订单金额**：未发现明显线性关系，但会员群体在高忠诚度区间订单金额更稳定

### 5. 时间维度分析
- **24小时订单分布**：呈双峰型，高峰在 11:00–13:00 和 18:00–20:00
- **12个月订单分布**：无明显季节性波动
- **节假日/周末 vs 平日**：节假日/周末的客单价均值（121.71）显著高于平日（111.94）→ 促销活动可在节假日重点投放

## 核心洞察与业务建议

| 洞察 | 建议 |
|------|------|
| 高峰时段（11-13点，18-20点）订单量集中，但延迟率上升 | 高峰时段启动动态溢价激励骑手，对用户承诺更长配送时间 |
| 备餐时间过长是取消订单的主要原因 | 开发“备餐超时预警”功能，主动推送优惠券安抚客户 |
| 会员客户客单价更高，但忠诚度评分与订单金额无强相关性 | 设计会员专享满减券或“准时保”服务，进一步提升客单价 |
| 三级城市配送效率最低 | 调整配送半径、增加运力或优化路线算法 |
